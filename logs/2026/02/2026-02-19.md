# Captain's Log — 2026-02-19
Day 50 of the year, and the work was spread across three different private projects — nothing thematically unified, just a lot of loose ends getting tied off.

The heaviest lift was in the first private project, where signup work began in earnest. Twelve files changed in that single PR tells me this wasn't a trivial form-and-submit job — likely involves auth plumbing, validation logic, maybe onboarding state management. Part 1 suggests there's more coming, which is usually a sign that someone made the right call not to cram everything into one review. Alongside that, I bumped the CI pipeline to Node 22 and synced the lockfile to match, then pinned the engine field in `package.json` to v22 as well. Keeping those two in sync matters — nothing worse than CI running a different Node version than what you've declared as a requirement, and having that bite someone in a subtle dependency resolution edge case weeks later. Node 22 is the current LTS, so this was overdue housekeeping.

The second private project saw a pair of routing cleanups. The first consolidated what sounds like a multi-environment API setup down to a single production endpoint — a small diff but meaningful signal that some earlier scaffolding has been retired. The second was a French localization route rename, swapping a dev-prefixed path for the clean production slug `/salle-de-presse`. That kind of leftover dev route living in production routing config is exactly the sort of thing that slips through and then causes confusion for months.

Over in the third private project, I enabled caching and wired up a `pressRelease` target transform for a press room feature. That's likely a content pipeline or CMS-adjacent layer where incoming data needs to be shaped before it hits the frontend — adding a named transform target keeps it composable and easier to extend later. Caching on top of that makes sense once the transform is stable; no point adding cache complexity before the shape of the data is settled.

Scattered day, but the kind of maintenance and foundation work that makes everything else easier down the road.
