# Captain's Log — 2026-02-04
Three commits today, split across two private projects — nothing dramatic, but genuinely useful groundwork.

The bulk of the focus was on a routing layer in the first project. Added rewrites to properly handle web components across development, production, and staging environments — the kind of configuration that's easy to overlook until something works locally and breaks in production. Getting environment-specific rewrite rules right matters more than it sounds; mismatched paths between environments create subtle failures that are annoying to diagnose. Alongside that, I wired up dynamic routes for a bilingual feature set, supporting both French and English. Internationalized routing always has more edge cases than you'd expect — slugs, canonical paths, language negotiation — so having the route definitions locked in early is the right call.

The second project had a small but important fix in the search layer. Somewhere along the way, the Algolia index name was being pulled from `import.meta.env` directly rather than going through the application's config abstraction. That's the kind of thing that works fine until it doesn't — `import.meta.env` is Vite-specific and can behave differently depending on build context or how the module is loaded. Centralizing environment access through a config object is cleaner, more testable, and makes the dependency explicit. Three files touched, but the change itself is tight and principled.

No PRs opened today, which means these are all sitting in local branches or pushed without review — probably iterating before they're ready to share. A quiet day technically, but the kind of work that keeps larger features from going sideways later.
