# Captain's Log — 2026-02-12
Three small but distinct commits today, split across two private projects.

Most of the day's meaningful work happened in private-project-2, where I wired up a new press room endpoint. The interesting bit wasn't just the REST API fetch — it was the CDN URL rewriting layer on top of it. Anytime you're pulling content from an API and then transforming asset URLs to point at a CDN rather than the origin, you need to be careful about edge cases: relative vs. absolute paths, protocol-relative URLs, and making sure you don't accidentally rewrite URLs that shouldn't be touched. Five files changed for what sounds like a simple feature suggests there was a fair amount of plumbing involved — probably a service or utility layer for the rewriting logic kept separate from the endpoint handler itself, which is the right call. Mixing URL transformation logic directly into a route handler is the kind of shortcut that causes pain later.

The navbar change, stripping years from event links, is a small but meaningful UX cleanup; year-scoped navigation tends to age poorly and creates dead links, so flattening that structure is usually the right long-term call.
