# Captain's Log — 2026-02-10
Three commits today, but they carry more weight than the count suggests.

The bulk of the thinking went into a bug fix on private-project-2 — a subtle but important authentication issue with the Firebase Admin SDK. When the expected environment variables aren't present, the SDK was failing hard instead of gracefully falling back to Application Default Credentials. ADC is exactly the right escape hatch here: it covers local development via `gcloud auth application-default login`, CI environments with workload identity, and Cloud Run without any explicit credential configuration. The fix is small — two files touched — but the kind of thing that causes real pain when it's missing. Someone running the project locally without a service account key would have hit an opaque initialization error. Now it degrades gracefully to the ambient credential chain, which is almost always what you want in a Google Cloud-native context.

On private-project-1, I advanced the submodules to their latest dev-branch state and added a `CLAUDE.md`. That second commit is worth noting — it's a conventions file for Claude Code, the Anthropic AI coding assistant, essentially a project-specific prompt that tells the model how to navigate the codebase, what patterns to follow, and what to avoid. I've been experimenting with using these as a forcing function for documenting implicit architectural decisions that never make it into a README. If the AI needs to be told the rules to be useful, the rules probably needed to be written down anyway.

Light day overall, but the Firebase fallback fix was the right thing to ship cleanly rather than leave half-done.
