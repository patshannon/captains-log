# Captain's Log — 2026-03-23
A day dominated by a B2B platform migration and a surprisingly annoying caching bug — two distinct problems that ended up touching four different codebases.

The meatier work was the migration of what looks like a professional partner portal ("espace pro") to a new external platform. The core of this was wiring up redirects and plumbing a new config value — `proSpaceDraftsUrl` — through to the footer navigation quit button, so users get handed off cleanly to the new external destination rather than hitting dead routes. Alongside that, I cleaned up the account UI: partner-related menu items are now hidden in standard (non-B2B) account navigation states, and the partner section was stripped from the profile desktop screen. The conditional is an `is-in-b2b-app` flag that gates all of this — it's a clean enough pattern for distinguishing B2B context from the default consumer experience. Two PRs merged to prod by end of day, which always feels satisfying.

The hotfix work ran in parallel across a couple of other projects — the same `is-in-b2b-app` flag needed to be threaded into draft and profile account wrappers that had been missed in the initial pass. Quick fix, but the kind of thing that's easy to overlook when a feature flag gets introduced in one place and you realize its blast radius is wider than you thought.

The caching bug was the most interesting in a "why is this broken" kind of way. The sitemap.xml was getting routed through SWR/KV caching, which treats responses as strings — fine for HTML or JSON, catastrophic for the binary XML output that sitemaps can produce in some build pipelines. The fix was surgical: explicitly exclude `/sitemap.xml` from the caching layer. It's the kind of one-line change that takes ten minutes to write and forty-five to diagnose. Needed to apply the same fix in two separate projects, which suggests the caching setup is shared infrastructure — worth noting in case other binary responses (feeds, downloadable assets) hit the same wall later.

Heavy day across a lot of surface area, but everything shipped.
