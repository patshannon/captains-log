# Captain's Log — 2026-03-05
Two distinct threads today, though both ended up circling the same underlying theme: getting things configured correctly so the actual work can happen.

Most of the frustration was in the infrastructure side of a private project. The app uses Sanity as its CMS, and somewhere along the way the Dockerfile was missing the necessary `ARG` declarations for Sanity's environment variables. Docker build args are easy to overlook — they need to be explicitly declared in the Dockerfile *and* passed through the build workflow, and if either side is missing, the build silently bakes in empty values or fails altogether. Fixed the Dockerfile first, then realized the CI build workflow had the same gap on its end. Two commits, one root cause: the build pipeline wasn't threading Sanity config through properly. Straightforward fix once you know where to look, but the kind of thing that can waste a surprising amount of time if you're not familiar with how Docker's ARG scoping works across multi-stage builds.

The more interesting work was on the LinkedIn bot project. Laid down a base resume generator script — the kind of thing that takes a structured configuration and produces formatted output in one or more target formats. The design decisions here matter a lot upfront: how opinionated should the configuration schema be, how flexible should the output options be, and how do you keep the two decoupled enough that you can add a new output format later without touching the core logic. Got a solid foundation down that I think threads that needle reasonably well.

That tied directly into a pull request on my resume repo — updated the content itself to reflect changes from the new base resume the generator is working from. Four files changed, which tells me the content is still spread across more places than I'd like. Part of the longer-term goal here is that the generator eventually becomes the single source of truth, so that kind of multi-file update becomes unnecessary. Not there yet, but today's work is a step in that direction.
